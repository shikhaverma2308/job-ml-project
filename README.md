# job-ml-project

# ML Job Recommendation System 🚀

## Overview
This project recommends jobs based on user skills using Machine Learning and NLP.

## Features
- Job Classification using ML
- Job Recommendation System
- Real-time Job Scraper
- Skill Gap Analyzer
- FastAPI Backend

## Tech Stack
Python, Scikit-learn, Pandas, FastAPI, Selenium

## How to Run
1. Install libraries: pip install pandas scikit-learn fastapi uvicorn selenium  
2. Run API: uvicorn app:app --reload  
3. Open browser: http://127.0.0.1:8000  

## Example
Input: Python Machine Learning  
Output: ML Job + Missing Skills
